pub fn with_num_threads(self, num_threads: usize) -> Option<Self>

Sets the number of threads to use for computing the noise contours.

This method will allocate if the threads are spawned.

Consuming method. Returns Some if the threads were successfully spawned, and None otherwise.

# Panics

Panics if num_threads is not a power-of-two value (required to efficiently divide the image buffer up).
