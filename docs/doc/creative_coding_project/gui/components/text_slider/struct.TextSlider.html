pub struct TextSlider { /* private fields */ }
Expand description

A simple slider with a text readout.

Holding shift whilst dragging the slider enables 10x fine value control; alt- or cmd-clicking the slider will reset it to its default value, if one is provided.

Scrolling over the value will also change it.

pub fn with_value_chars(self, num_chars: usize) -> Self

Sets the number of chars used to show the value read out. The default value is 5. This includes any decimal points, but ignores any prefix or suffix. Does not apply if the slider uses integer rounding.

Note that this value is not strictly adhered to, and the text formatting will always allow the whole value to be shown, and always includes 1 decimal value if the value would otherwise be truncated at the decimal point.

See with_max_value_num_chars() to have use a constant number of chars.
